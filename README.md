# Sales Analytics Dashboard

A modern, interactive sales dashboard built with Next.js 15, TypeScript, and Tailwind CSS. Features multiple chart visualizations, data filtering, and year-over-year sales analysis.

## Features

- **Multi-Year Analysis**: Compare sales performance across 2022, 2023, and 2024
- **Interactive Charts**: Switch between bar, line, and pie charts using Recharts
- **Custom Filters**: Set sales thresholds to filter and analyze data
- **Responsive Design**: Fully responsive layout that works on all devices
- **Mock Data**: Realistic sales data based on Kaggle datasets
- **Type Safety**: Full TypeScript support with typed components and data

## Tech Stack

- **Next.js 15**: React framework with App Router
- **TypeScript**: Type safety and better developer experience
- **Tailwind CSS**: Utility-first CSS framework
- **Recharts**: Composable charting library
- **Lucide React**: Beautiful icons
- **Vercel**: Deployment platform
