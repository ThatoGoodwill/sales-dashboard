#  Sales Dashboard – Next.js

A modern Sales Dashboard web application built with **Next.js**, **Tailwind CSS**, and deployed on **Vercel**.  
This project was developed as part of an internship technical assessment to demonstrate frontend structure, UI design, and deployment.
